### Dotfiles for my hyprland, waybar, rofi setup

### Main Dependencies:
- [Hyprland](https://github.com/hyprwm/Hyprland)
- [Rofi](https://github.com/davatorium/rofi)
- [Waybar](https://github.com/Alexays/Waybar)
- [Kitty](https://github.com/kovidgoyal/kitty)

### Using Catppuccin for theming:
- [Catppuccin Rofi](https://github.com/catppuccin/rofi)
- [Catppuccin Waybar](https://github.com/catppuccin/waybar)
- [Catppuccin Kitty](https://github.com/catppuccin/kitty)
- [Catppuccin NVim](https://github.com/catppuccin/nvim)
